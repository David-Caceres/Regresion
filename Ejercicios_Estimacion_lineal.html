## (Intercept)         lpsa  
##     -0.5086       0.7499

Para comparar ambas rectas tenemos que invertir una de ellas, ya que si tomamos la ecuacin de la recta, y= a + b * x para la recta1, para la recta2 la x y la y deben intercambiarse, de ese modo:

x = a + b * y y = (x - a/b) y = -a/b + 1/b * x

plot(lpsa ~ lcavol, data =prostate)
abline(recta1, col="red")
a_2<-recta2$coefficients[1]
b_2<-recta2$coefficients[2]
a2<- -a_2/b_2
b2<- 1/b_2
abline(a2,b2, col="blue")
